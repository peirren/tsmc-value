靜態網頁版估值模型

台積電 EPS × 成長率 × PEG 估值模型

這個版本是把你 Excel 裡的邏輯改寫成純前端網頁：只要輸入 EPS、成長率、基準年度與預測年數，頁面就會自動算出 PEG 對應本益比、目標價，以及未來幾年的 EPS / 股價區間。

輸入區

這一區就是把 Excel 裡的可調參數拉出來，讓你在網頁上直接改。

這段在做什麼：設定估值的起點 EPS。
這段在做什麼：設定 EPS 年成長率，例如 25% 就填 0.25。
這段在做什麼：設定預測表第一年的年份。
這段在做什麼：控制你要往後看幾年。
這段在做什麼：定義保守 / 基準 / 樂觀三種情境。
這段在做什麼：建立年度估值矩陣，觀察不同 PE 下的價格範圍。
這段在做什麼：把 Excel 的三個核心公式明文化，避免變成黑盒。 PE = 成長率 × 100 × PEG 目標價 = EPS × PE 未來 EPS = 當年 EPS × (1 + 成長率)

模型摘要

這一區用最少的數字，先給你結論。

PEG 對應估值表

這段在做什麼：把「成長率 → 合理 PE → 目標價」直接展開成情境表。

年度 EPS / 股價矩陣

這段在做什麼：把每年 EPS 預測，乘上不同 PE，形成多情境股價區間。

人類能懂的系統流程

1. 使用者輸入參數
輸入目前 EPS、預期成長率、年份與估值情境。
2. 前端在瀏覽器內運算
JavaScript 直接照公式計算，不需要後端、不需要資料庫。
3. 產出估值結果
同時顯示 PEG 估值表與年度估值矩陣。
4. 部署成靜態網站
把這一個 HTML 檔丟到 GitHub Pages，就能公開存取與分享。
